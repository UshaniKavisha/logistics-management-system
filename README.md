📦 Logistics Management System
📌 Description
This project is a simple Logistics Management System developed using the C programming language. It is a menu-driven program that helps manage cities, delivery routes, vehicles, and customer orders.

🎯 Objectives
Manage cities and distances
Handle delivery requests
Calculate delivery cost and time
Find the shortest route
Generate performance reports

⚙️ Features
Add, rename, remove, and list cities
Store distances using 2D arrays
Select vehicles (Van, Truck, Lorry)
Process delivery requests
Calculate cost, time, and fuel
Find least-cost (shortest) route
Generate reports (total deliveries, profit, distance)
Save and load data using files

🧠 Concepts Used
Arrays
Functions
Loops
Conditional statements
File handling

🖥️ How to Run
Open the project in Code::Blocks or any C compiler
Compile the program
Run and use the menu options

📊 Output
The system shows:
Menu options
City and distance details
Delivery results
Performance report
